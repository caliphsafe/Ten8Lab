# 10×8 Creative Industry Lab — Pitch Website

A single-page, 12-section interactive presentation website proposed for The Zeiterion, sponsors, foundations and local officials.

## Files

- `index.html`
- `style.css`
- `script.js`

## GitHub → Vercel

1. Create a new GitHub repository.
2. Upload the three website files to the repository root.
3. In Vercel, import the GitHub repository.
4. Framework preset: **Other**.
5. Build command: leave blank.
6. Output directory: leave blank.
7. Deploy.

## Notes

- No framework or package installation is required.
- The website is responsive and supports keyboard navigation.
- Replace the email link in `index.html` if the proposal contact changes.
- Partnership naming and brand usage are presented as a proposal and remain subject to approval.
